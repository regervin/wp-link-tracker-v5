== Description ==

WP Link Tracker is a comprehensive link shortening and tracking solution for WordPress. It allows you to create shortened links, track clicks with detailed analytics, and manage your marketing campaigns effectively.

### Features

* **Link Shortening**: Create short, branded links for your marketing campaigns.
* **Click Tracking**: Track every click with detailed information about visitors.
* **Detailed Analytics**: View comprehensive statistics including clicks, unique visitors, referrers, and more.
* **Campaign Management**: Organize your links by campaigns for better tracking.
* **Device Tracking**: See which devices, browsers, and operating systems your visitors are using.
* **UTM Parameter Support**: Add and track UTM parameters for better campaign analysis.
* **Shortcode Support**: Easily embed tracked links in your posts and pages.
* **Dashboard Overview**: Get a bird's-eye view of your link performance.

### Use Cases

* **Marketing Campaigns**: Track the effectiveness of your marketing efforts across different channels.
* **Email Marketing**: See which links in your emails are getting the most clicks.
* **Social Media**: Compare performance across different social platforms.
* **Affiliate Marketing**: Track which affiliate links are performing best.
* **A/B Testing**: Test different call-to-actions or landing pages.

== Installation ==

1. Upload the `wp-link-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Link Tracker' in your admin menu to start creating tracked links

== Frequently Asked Questions ==

= How do I create a tracked link? =

Go to Link Tracker > Add New in your WordPress admin, enter a title and destination URL, then publish. The plugin will automatically generate a short URL for you.

= Can I customize the short URL format? =

Yes, you can change the prefix used in your short URLs (default is "go") in the plugin settings.

= Does this plugin work with any theme? =

Yes, WP Link Tracker works with any properly coded WordPress theme.

= Can I track UTM parameters? =

Yes, the plugin captures UTM parameters and provides analytics based on them.

= Is there a limit to how many links I can track? =

No, you can create and track as many links as you need.
